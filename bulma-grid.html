The is-multiline Class

By default, Bulma tries to fit all columns in one row. Adding is-multiline allows them to wrap to the next line.

<div class="columns is-multiline">

Grid Math (is-4)

Bulma uses a 12-column system. 12 / 4 = 3. So, is-4 creates 3 equal columns per row.

<div class="column is-4">

Why did this wrap?

Since the first three boxes already took up all 12 slots (4+4+4), the 4th box has no room and "wraps" to the next row automatically!
