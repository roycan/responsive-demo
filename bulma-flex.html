The Container

In Bulma, you wrap everything in a columns div. This activates Flexbox.

<div class="columns">
  </div>

The Individual Item

Each item inside uses the column class. Bulma divides the space equally by default.

<div class="column">
  Content here
</div>

Built-in Responsiveness

Bulma is Mobile-First. This means columns automatically stack on phones and turn into a row on tablets/desktops without you writing any CSS!

/* No CSS needed! 
   Bulma handles the 
   @media queries for you. */
